# Idea Platform Generator

Генератор структуры песни для REAPER. Создаёт случайную форму композиции (темп, размер, тональность, лад, секции) и сразу раскладывает её на таймлайн: регионы, маркеры тональности и темпо-маркеры. Полностью интерактивный GUI на ReaImGui.

Принцип управления: **«включено» = шанс больше 0, «выключено» = шанс 0**. Никакого выделения мышкой — всё задаётся числами-весами.

---

## Возможности

### Управление шансами
- **Два режима** (глобальный переключатель):
  - **Шансы** — везде поля с вероятностями (0–100).
  - **Редактирование** — добавление, переименование, удаление и перестановка значений.
- **4 колонки значений:**
  - **Темп** — диапазон 60–240 BPM.
  - **Размер** — от 4/4 до 13/8 (14 вариантов по умолчанию, можно добавлять свои).
  - **Тоника** — 12 нот.
  - **Лад** — 16 ладов (натуральные, пентатоники, гармонические, мелодические, целотонный, уменьшённые, хроматический).
- Каждое значение имеет **вес**. Чем выше вес — тем чаще значение выпадает. `0` = выключено.

### Форма композиции
Таблица секций (Интро, Куплет, Припев и т.д.) с параметрами:

| Параметр | Назначение |
|---|---|
| **Тактов** | Длина в тактах. Поддерживает списки и диапазоны: `8`, `4,8,16`, `4-8`, `4,6-8,16` — значение выбирается случайно из набора. |
| **Появл.** | Шанс появления секции в данной генерации (0–100). |
| **Доп.такты** | Шанс добавить 1–2 дополнительных такта. |
| **ΔТемп / ΔРазмер / ΔТоника / ΔЛад** | Шанс смены значения относительно предыдущей секции. `0` — наследуется, `100` — всегда новое значение. |
| **Линк** | Связывает секции одной группой (одинаковый номер). Все залинкованные секции повторяют первую появившуюся в группе один-в-один (темп, размер, тональность, лад, длина). |

### Генерация и вывод на таймлайн
- **Регионы** — только название секции (структура формы), без технических данных.
- **Маркеры тональности** (точечные) — `Тоника Лад`, ставятся один раз при смене (новизне). Первая секция всегда даёт маркер.
- **Темпо-маркеры** — темп и размер, без текстового дублирования в регионах.
- **Цвет генерации** — каждая генерация получает один случайный цвет, им красятся все её регионы и маркеры. Разные «песни» визуально различимы на таймлайне.
- Генерация создаёт регионы, начиная от позиции курсора; старые маркеры на этой позиции удаляются.

### Пресеты
- Сохранение / загрузка / удаление снимков всех чисел по имени.
- Загрузка пресета — по клику на имя в списке (без отдельной кнопки).
- Удаление — кнопкой «Удалить» для выбранного.

### Сохранение состояния
- Вся форма и все шансы автосохраняются при изменениях и при закрытии окна.
- Состояние полностью восстанавливается при следующем запуске (включая добавленные и удалённые секции/значения).

### Отладочный дамп
- После каждой генерации в ReaScript console выводятся полные настройки (шансы, форма с раскрытыми наборами тактов) и итог (развёртка всех созданных секций с позициями, темпом, тональностью).

## Установка

### Требования
- **REAPER** (любая актуальная версия).
- **[ReaImGui](https://github.com/cfillion/reaimgui)** версии **0.10 или выше** (требуется для корректного отображения кириллицы).

### Установка ReaImGui
1. Открыть в REAPER: **Extensions → ReaPack → Browse packages**.
2. Найти `ReaImGui` и установить.
3. Если ReaPack отсутствует — установить его с [reapack.com](https://reapack.com/).

### Установка скрипта
1. В REAPER: **Actions → Show action list → New action → Load REAscript**.
2. Выбрать файл `IdeaPlatformGen2.lua`.
3. Запустить действие (двойной клик или назначить горячую клавишу).
4. При первом запуске откроется окно генератора.

Скрипт не требует дополнительной настройки — состояние хранится во встроенном хранилище REAPER (`ExtState`) и переносится между запусками.

## Использование

1. Настроить веса значений в колонках (Темп, Размер, Тоника, Лад) — включить нужные, выключить ненужные (шанс 0).
2. Настроить форму: задать секции, длины, шансы появления и Δ-параметры.
3. При необходимости связать повторяющиеся секции линком (например, все припевы — линк 2).
4. Поставить курсор на нужную позицию таймлайна.
5. Нажать **СГЕНЕРИРОВАТЬ** — регионы и маркеры создаются от позиции курсора.
6. Для просмотра деталей — открыть **ReaScript console** (View → ReaScript console): там полный дамп настроек и результата.
